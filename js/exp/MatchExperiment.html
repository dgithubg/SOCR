Description

The experiment is to randomly permute \(n\) balls, numbered 1 to \(n\). A match occurs whenever the ball number and the position number agree. The matches are shown in red. Therandom variable \(N\) that give the number of matches is recorded on each update. The probability density function and moments of \(N\) are shown in blue in the distribution graph and are recorded in the distribution table. On each update, the empirical density function and moments are shown in red in the distribution graph and are recorded in the distribution table. The parameter \(n\) can be varied with the input control.
