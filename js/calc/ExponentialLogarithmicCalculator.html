Exponential-Logarithmic Distribution Calculator

Distribution graph

Description

This applet is a calculator for the exponential-logarithmic distribution with shape parameter \( p \) and scale parameter \( b \). The calculator gives the value of the cumulative distribution function \(p = F(x)\) for a given value of \(x\), or the value of the quantile function \(x = F^{-1}(p)\) for a given value of \(p\). The shape parameter \(p\) and and the scale parameter \( b \) can be varied with the input control, as can the variables \(x\) and \(p\).
