Discrete Uniform Distribution Calculator

Distribution graph

Description

For the discrete uniform distribution, this calculator gives the value of the cumulative distribution function \(p = F(x)\) for a given value of \(x\), or the value of the quantile function \(x = F^{-1}(p)\) for a given value of \(p\). The left endpoint \(a\) and the number of points \( n \) can be varied with the input control, as can the variables \(x\) and \(p\).
