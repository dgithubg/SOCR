Pareto Distribution Interactive Calculator

Distribution graph

Description

For the Pareto distribution, this calculator gives the value of the cumulative distribution function \(p = F(x)\) for a given value of \(x\), or the value of the quantile function \(x = F^{-1}(p)\) for a given value of \(p\). The shape parameters \(k\) and scale parameter \(b\) of the distribution can be varied with the input controls, as can the variables \(x\) and \(p\)
